# FXHASH EVM SDK

## Setup

1. Clone repository

```
git clone git@github.com:fxhash/fxhash-evm-contracts.git
```

2. Deploy contracts from the `contracts` folder

```
npm run deploy
```

3. Create `.env` file in `sdk`` root directory

```
CONTRACTS_PATH=
PROVIDER_URL=
PROJECT_FACTORY_ADDRESS=
SIGNER_PK=
```

3. Install dependencies

```
npm i
```

4. Run prettier

```
npm run prettier
```
